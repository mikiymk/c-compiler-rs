# 出来ること

- 0 ～ 255の数を返す
- 足し算+、引き算-
- 掛け算*、割り算/、()による計算の優先順位
- 単項プラス+、単項マイナス-
- 比較(等しい==、等しくない!=、大小<>)
- 区切られた複数の文;
- 26個までの変数(1文字、複数文字)
- return文
- if文、while文、for文
- {}による複数文の実行
- 外部関数の呼び出し(引数なし、引数1こ～6こ)
- 関数の定義(引数なし、引数1こ～6こ)
- アドレスを取得&とアドレスの中身を参照*
